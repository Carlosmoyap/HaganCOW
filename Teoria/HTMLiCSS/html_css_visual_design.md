# HTML & CSS Presentació Visual Design Scheme

## 1. Design Philosophy

### Overall Visual Direction

- Un disseny acadèmic net i estructurat que reflecteix la precisió del codi. Inspirat en la claritat de Muji i la funcionalitat de Dieter Rams, utilitzant una estructura de quadrícula implícita per organitzar el contingut dens de manera accessible. El disseny prioritza la legibilitat i la jerarquia visual per facilitar l'aprenentatge.

### Content Density

- **Densitat alta**: Aquesta presentació és un resum acadèmic d'1 hora, amb alta densitat de contingut tècnic, incloent sintaxi de codi, etiquetes i propietats. Cada diapositiva hauria de contenir 3-5 elements clau, prioritzant la claredat i l'eficiència de la informació.

## 2. Color System

**Colors Primaris**

- Color de fons: #1A1D23 (fons fosc per a codi)
- Color primari: #5C7A96 (blau grisenc acadèmic)
- Color secundari: #8A9BA8 (gris blavós per a suport)
- Color d'accentuació: #D4A26A (taronja suau per a destacar)

**Colors de Text**

- Text fosc (per a fons clars): #2D313A
- Text clar (per a fons foscos): #E0E2E6

## 3. Font Design

**Fonts de Títol**

- Font anglesa/ciríl·lica
  - Chiron Hei HK (Un tipus de lletra sans-serif modern i geomètric, la seva forma neta transmet un fort sentit tecnològic)
  - Quattrocento Sans (Un sans-serif clàssic i còmode, la seva forma oberta i clara és molt llegible, adequada per a títols acadèmics)

**Fonts de Cos**

- Font anglesa/ciríl·lica
  - Quattrocento Sans (Un sans-serif clàssic i elegant, altament llegible en mides petites, perfecte per a text dens de codi i explicacions)
  - Sorts Mill Goudy (Un serif clàssic amb un toc acadèmic, proporcionant una sensació de llibre de text tradicional, adequat per a paràgrafs descriptius)

## 4. Image Style

- Les imatges haurien de ser abstractes i conceptuals, utilitzant formes geomètriques, línies de connexió i representacions de xarxes per simbolitzar estructures i relacions. Quan es mostrin captures de pantalla de codi, utilitzar un tema de colors fosc (Dark Mode) amb sintaxi ressaltada per coherència i millor legibilitat.
