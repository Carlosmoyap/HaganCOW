Introducció a Bootstrap

El framework HTML, CSS i JS per a projectes responsive.

Basat en: 01_6_BOOTSTRAP.pdf

Instal·lació: Evolució (v3 vs v5)

Bootstrap 3 (PDF)

Necessita jQuery per funcionar.

<!-- CSS -->
<link href=".../bootstrap.min.css" rel="stylesheet">

<!-- JS (Ordre importància!) -->
<script src=".../jquery.min.js"></script>
<script src=".../bootstrap.min.js"></script>
Dependència: Sense jQuery, els menús i modals no funcionen.

Bootstrap 5 (Actual)

Sense jQuery. Usa JavaScript natiu.

<!-- CSS -->
<link href=".../bootstrap.min.css" rel="stylesheet">

<!-- JS (Bundle inclou Popper) -->
<script src=".../bootstrap.bundle.min.js"></script>
Millora: Més lleuger i ràpid al no dependre de llibreries externes grans.

Disseny Responsive

Per garantir que les pàgines es visualitzin i comportin correctament en dispositius mòbils, és obligatori afegir l'etiqueta viewport dins del <head>.

<meta name="viewport" content="width=device-width, initial-scale=1">

width=device-width: Ajusta l'ample de la pàgina a l'ample de la pantalla del dispositiu.

initial-scale=1: Estableix el nivell de zoom inicial quan es carrega la pàgina.

Nota sobre versions: Aquest requisit és universal. Tant Bootstrap 3, 4 com l'actual Bootstrap 5 necessiten aquesta línia. Sense ella, els mòbils renderitzaran la web com si fos una pantalla d'escriptori (lletra minúscula).

Taules

Afegint la classe .table es formateja automàticament.

<table class="table table-striped table-bordered">
  <thead>
    <tr>
      <th>#</th>
      <th>Nom</th>
      <th>Cognom</th>
    </tr>
  <tbody>
    <tr>
      <td>1</td>
      <td>Mark</td>
      <td>Otto</td>
    </tr>
    <tr class="success">
      <td>2</td>
      <td>Jacob</td>
      <td>Thornton</td>
    </tr>
  </tbody>
</table>
