Llenguatges de Programació Web

Costat Client

S'executen al navegador de l'usuari. No accedeixen directament a la BD.

  • HTML: Estructura (Hypertext Markup Language).
  • CSS: Estil i disseny.
  • JavaScript: Interactivitat.
  • Applets Java: (En desús) Petits programes incrustats.
  • VBScript: (En desús) Scripting de Microsoft.

Costat Servidor

S'executen al servidor web. Generen l'HTML dinàmicament.

  • PHP: El més popular en entorns Linux/Apache.
  • ASP / ASP.NET: Tecnologia Microsoft.
  • JSP (Java Server Pages): Tecnologia Java.
  • Perl: Molt potent per processament de text (CGI).
  • Python / Ruby: Moderns i elegants.

Conceptes: Frameworks

Definició

Conjunt estandarditzat de conceptes, pràctiques i criteris per enfocar un tipus de problemàtica. Serveix de referència per organitzar i desenvolupar software.

Avantatges

  • Evita codi repetitiu.
  • Gestió d'errors i seguretat integrada.
  • Redueix costos i temps.

Exemples PHP

  • Zend Framework: Orientat a objectes, MVC, suport BD (Oracle, DB2, MySQL).
  • Symfony: Rapidíssim, sistema de plantilles, caché, motor BD intel·ligent.
  • CakePHP: Estructura MVC, compatible PHP4/5.
  • CodeIgniter: Molt ràpid i lleuger.
  • Phalcon: Escrit com extensió C per màxim rendiment.
  • Laravel: Sintaxi elegant i expressiva.

Entorn Local: XAMPP

Què inclou?

Apache + MySQL + PHP + Perl.

Instal·lació

  • Descarregar de apachefriends.org.
  • Executar instal·lador (Next > Next).
  • Directori recomanat: c:\xampp.
  • Panell de Control: Permet arrencar/aturar serveis (Apache, MySQL).
Important: Si Skype utilitza el port 80, Apache no arrencarà. Cal canviar la configuració de ports.

Estructura de Carpetes

c:\xampp\
├── apache\ (Configuració servidor)
├── php\ (Configuració llenguatge)
├── mysql\ (Dades BD)
└── htdocs\ (Directori WEB)
    └── el_meu_projecte\
        └── index.php

Per veure la web, obrir al navegador: http://localhost/el_meu_projecte/

Configuració PHP: php.ini

Fitxer de configuració ubicat a C:/xampp/php/php.ini. Paràmetres clau:

short_open_tag = On Permet utilitzar etiquetes curtes <? ?>.
safe_mode = Off Habilita totes les funcions PHP (recomanat per educació).
display_errors = On Mostra errors al navegador (vital per depurar).
max_execution_time = 30 Segons màxims d'execució d'un script.
memory_limit = 128M Memòria màxima per script.
error_reporting = E_ALL & ~E_NOTICE... Nivell de detall dels errors mostrats.
post_max_size = 8M Màxim de dades enviades per POST.
upload_max_filesize = 8M Mida màxima per fitxers pujats.
file_uploads = On Habilita la pujada de fitxers via HTTP.
SMTP = localhost Configuració per enviament de correus (mail function).

Configuració Apache: .htaccess

Fitxer ocult que permet configuració descentralitzada per directori. Capacitats:

  • Autorització i Autenticació: Protegir carpetes amb contrasenya (`.htpasswd`).
  • Pàgines d'Error: Personalitzar missatges 404, 500.
  • Llistat de Directoris: Activar/Desactivar l'índex de fitxers.
  • MIME Types: Definir com el servidor tracta certs fitxers.
  • Cache Control: Gestionar la memòria cau del navegador.
  • CGI Handlers: Executar scripts Perl/Python.

Exemple pràctic:

# Evitar llistar fitxers
Options -Indexes

# Pàgina 404 personal
ErrorDocument 404 /errors/notfound.html

# Redirecció 301
Redirect 301 /vella.php /nova.php

# Protecció de fitxer específic
<Files "config.php">
  Order Allow,Deny
  Deny from all
</Files>
